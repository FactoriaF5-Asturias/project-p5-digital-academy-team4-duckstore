# 🦆 DuckStore - Proyecto de Desarrollo Web

![DuckStore Banner](https://via.placeholder.com/800x200?text=DuckStore+Project+Design)

## 🌟 Sobre el Proyecto
Este proyecto consiste en la creación de la plataforma digital para **DuckStore**, una tienda especializada en patitos de colección. El trabajo abarca desde la conceptualización visual y arquitectura de información hasta el desarrollo técnico final, asegurando una experiencia de usuario intuitiva y moderna.

---

## 🛠️ Tecnologías y Herramientas

### **Desarrollo Core**
* **HTML5:** Uso de etiquetas semánticas para una estructura sólida y accesible.
* **CSS3:** Implementación de layouts avanzados (Flexbox/Grid), animaciones, transiciones y diseño responsivo.

### **Diseño y Planificación**
* **Figma:** Creación de *wireframes* de baja fidelidad y *mockups* de alta fidelidad.
* **Stitch:** Prototipado y flujo de usuario.

### **Entorno de Trabajo**
* **Visual Studio Code:** Editor de código.
* **GitHub Pages:** Hosting y despliegue del proyecto en producción.

---

## 📋 Requisitos y Funcionalidades

La web ha sido desarrollada siguiendo estrictamente los requisitos funcionales del cliente:

1. **Landing Page:** Presentación oficial de la tienda y su propuesta de valor.
2. **Catálogo de Productos:** Visualización de todos los patitos disponibles, incluyendo nombre, imagen y precio.
3. **Detalle del Producto:** Página específica con descripción detallada, precio, imagen y stock disponible.
4. **Página de Contacto:** Incluye la localización física de la tienda y un acceso directo para contactar vía email.

---

## 📐 Proceso de Trabajo

Para garantizar un resultado profesional, el desarrollo se dividió en las siguientes fases:

* **Arquitectura (Wireframes):** Definición de la estructura básica del sitio sin elementos visuales.
* **Prototipado (Mockups):** Diseño visual final y flujo de navegación interactivo en Figma.
* **Desarrollo Semántico:** Traducción del diseño a código HTML5 limpio.
* **Estilismo y Accesibilidad:** Aplicación de CSS3 para replicar el diseño exacto de los mockups, garantizando que el sitio sea accesible para todos los usuarios.
* **Adaptabilidad (Responsive):** Optimización total para dispositivos móviles, tablets y ordenadores de sobremesa.
